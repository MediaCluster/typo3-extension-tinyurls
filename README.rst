This is a TYPO3 Extension for converting a normal URL to a tiny URL.

It works similar as services provided by https://bitly.com/ or https://tinyurl.com/.

Have a look in the :code:`Documentation` folder for more information or
browse to these URLs:

* `TYPO3 Extension Repository (TER) <https://typo3.org/extensions/repository/view/tinyurls>`_
* `tinyurl Extension Manual <https://docs.typo3.org/typo3cms/extensions/tinyurls/>`_
